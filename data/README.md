# Данные проекта

## data/test_cases/eval_set.json

**20 тест-кейсов** для оценки качества AI-агента (baseline vs improved).

Каждый кейс содержит:
- `resume`: краткое резюме кандидата
- `job_description`: описание вакансии
- `question_type`: тип вопроса (behavioral / technical / situational)
- `candidate_answer`: ответ кандидата (сильный, средний, слабый или нерелевантный)
- `expected_score`: ожидаемая оценка (1-5)
- `expected_feedback_contains`: ключевые слова, которые должны быть в фидбеке

Покрытие сценариев:
- Junior / Middle / Senior уровни
- Frontend, Backend, DevOps, QA, ML, Design, PM, Analyst
- Оценки от 1 до 5 (по 3–4 кейса на каждый балл)
- Пограничные случаи: ответ-клише, ответ на другом языке, копипаста, слишком короткий ответ

## data/raw/interview_qa_examples.json

**30 примеров** вопросов и ответов с собеседований. Использовались для разработки few-shot примеров в промптах.

Каждый пример содержит:
- `question`: вопрос интервьюера
- `good_answer`: эталонный хороший ответ
- `bad_answer`: пример слабого ответа
- `evaluation_criteria`: критерии, по которым оценивается ответ

Данные собраны вручную на основе реальных практик IT-собеседований.
