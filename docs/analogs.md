# Анализ аналогов

| Аналог | Что умеет | Где использует ИИ | Сильные стороны | Слабые стороны | Цена / модель | Чем мы лучше (УТП) |
|--------|-----------|-------------------|-----------------|----------------|---------------|---------------------|
| **Pramp** | Mock-интервью с живыми людьми, peer-to-peer формат | Не использует ИИ для проведения интервью | Реальная практика с человеком, бесплатно | Зависимость от расписания партнёра, нет персонализации под вакансию, нет аналитики ответов | Бесплатно | AI доступен 24/7, адаптируется под конкретную вакансию и резюме |
| **Interviewing.io** | Анонимные mock-интервью с инженерами из FAANG | Минимально - запись и транскрипция | Реальные интервьюеры из топ-компаний | Дорого ($100+/сессия), ограниченное расписание | $100-225/сессия | Бесплатно, мгновенный доступ, детальная аналитика по компетенциям |
| **ChatGPT / Claude (прямой вызов)** | Можно попросить провести интервью в чате | Базовый LLM без адаптации | Быстро, бесплатно (с подпиской) | Нет анализа резюме, нет адаптации вопросов, нет структурированной оценки, нет аналитики | Подписка $20/мес | Агентный пайплайн с routing, контекст резюме+вакансии, структурированная оценка, визуальная аналитика |
| **Final Round AI** | AI-копилот для реальных собеседований (подсказки в реальном времени) | LLM для генерации подсказок | Помощь в реальном времени | Этические вопросы (обман работодателя), не учит готовиться самостоятельно | $96/мес | Фокус на обучении, а не на подсказках; прозрачная оценка навыков |

*Прямые конкуренты:* Pramp, Interviewing.io - сервисы mock-интервью.

*Косвенные конкуренты:* ChatGPT/Claude напрямую, Final Round AI - AI-инструменты без специализации на тренировке.
