# MoSCoW + MVP

## MoSCoW

| Категория | Функция | Описание | Кто делает |
|-----------|---------|----------|------------|
| **Must** (обязательно для MVP) | 1. Загрузка резюме | Парсинг PDF/DOCX, извлечение навыков и опыта | Backend |
| | 2. Ввод вакансии | Текстовое поле для описания вакансии | Frontend |
| | 3. AI-интервью | Чат с адаптивными вопросами (5-8 шт), routing по типам | AI-инженер |
| | 4. Оценка ответов | Chain-of-thought анализ каждого ответа | AI-инженер |
| | 5. Итоговый отчёт | Общий балл, фидбек по каждому ответу | Frontend + AI |
| **Should** (очень желательно) | 1. Радар-чарт компетенций | Визуализация сильных/слабых сторон | Frontend |
| | 2. История интервью | Сохранение и просмотр прошлых сессий | Backend + Frontend |
| | 3. Выбор типа собеседования | Behavioral / Technical / Mixed | Frontend + AI |

| | 2. Экспорт отчёта в PDF | Скачивание результатов | Backend + Frontend |

| | 3. Мультиязычность | Интерфейс на нескольких языках | - |

## MVP

Минимальный набор функций для демо:

1. Загрузка резюме (PDF/DOCX) с автоматическим извлечением ключевой информации
2. Ввод описания вакансии
3. Адаптивное mock-интервью в чате (5-8 вопросов с routing по типам)
4. Оценка каждого ответа с детальным фидбеком
5. Итоговый отчёт с аналитикой по компетенциям
